Проект представляет собой небольшое front-end приложение сервиса для метча людей по общим вкусам в кофе.

Всего будет 5 страниц:
    - страница входа/регистрации;
    - страница с метчами;
    - страница с каталогом кофейных напитков;
    - страница отдельного кофейного напитка;
    - страница профиля пользователя;
    
Из библиотек приложение глобально используёт библиотеку компонентов ant-design, библиотеку навигации react-router и библиотеку хранения данных zustand.
